Topic 24: fork() System Call

What is fork()?

The fork() system call is used in Unix/Linux operating systems to create a new process. The original process is called the parent process, and the new process is called the child process.

After fork(), both parent and child continue running from the next line of code. They are separate processes and each one has its own process ID.

Return Values

Return Value Meaning
0 The code is running inside the child process
Positive number The code is running inside the parent process
-1 fork() failed

Simple C Example

#include <stdio.h>
#include <unistd.h>

int main() {
    pid_t pid = fork();

    if (pid == 0) {
        printf("This is the child process\n");
    }
    else if (pid > 0) {
        printf("This is the parent process\n");
    }
    else {
        printf("Fork failed\n");
    }

    return 0;
}

Explanation

  • Before fork(), there is only one process.
  • When fork() is called, the operating system creates a child process.
  • The parent receives the child PID.
  • The child receives 0 from fork().
  • Parent and child run independently.

Priority Scheduling (Preemptive)

Preemptive Priority Scheduling is a CPU scheduling algorithm where the process with the highest priority gets executed first. If a new process arrives with a higher priority than the currently running process, the CPU immediately switches to the new process. This switching is called preemption.

In this demo, smaller priority numbers mean higher priority.
